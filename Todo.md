## 운영 규칙

### 1. 추가 규칙
- 모든 요청은 Todo에 먼저 기록한다.
- 리뷰 결과는 반드시 Todo로 연결한다.
- 중복 항목 생성 금지 → 기존 항목 업데이트

### 2. 분류 규칙
- blocker/high → 🔥 Critical
- 보안 문제 → 🔒 Security
- 구조 개선 → 🧠 Backlog
- 테스트 필요 → 🧪 Verification Needed

### 3. 완료 처리
- 완료 시 `[x]` 체크
- 필요 시 완료 섹션으로 이동

### 4. 세션 종료 전
- Todo 상태 최신화
- 진행/미완료 작업 정리
